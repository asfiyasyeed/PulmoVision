# PulmoVision
PulmoVision is a software-based AI medical image diagnosis system that uses Deep Learning to detect and classify pneumonia from chest X-ray images. The system integrates a React frontend, Node.js backend, and a Python-based AI microservice to provide automated and accurate diagnostic support.
